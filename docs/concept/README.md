# Dokumen Konsep Proyek

Folder ini berisi dokumen konsep awal proyek Indonesia Power Map.

## File

- **`01-project-concept.docx`** — Dokumen konsep awal disusun oleh Sandro Agassi
  pada awal 2026. Berisi visi proyek, masalah yang ingin diselesaikan, prinsip desain,
  roadmap 5 tahap (Step 1–5), rencana persiapan data, sumber data publik, tech stack
  awal, dan struktur folder. Inilah landasan filosofis dan teknis seluruh proyek.

## Tujuan folder

Folder ini berfungsi sebagai **catatan terversi dari evolusi konsep proyek**.
Setiap revisi besar dokumen konsep akan ditambahkan dengan nomor urut
(02-..., 03-...), tidak menggantikan versi lama. Ini penting untuk:

1. **Bukti originalitas** — dokumen tahap awal dengan timestamp Git commit menjadi
   prior art yang sulit dipalsukan.
2. **Transparansi proses berpikir** — pembaca/kontributor bisa melihat bagaimana
   konsep berkembang dari ide awal sampai implementasi.
3. **Reproducibility** — keputusan desain saat ini selalu bisa dilacak kembali ke
   versi konsep yang relevan.
